# City Screensaver

A terminal-based ASCII art screensaver that displays an animated cityscape at night.

## Features

-   Randomly generated cityscape with buildings of various heights and widths.
-   Building windows that randomly turn on and off.
-   Vehicles (cars, vans, and lorries) that drive by at street level.
-   A moon and twinkling stars in the night sky.
-   Written in Rust for performance and safety.

## Prerequisites

-   [Rust](https://www.rust-lang.org/tools/install)

## Building and Running

1.  Clone the repository:
    ```bash
    git clone https://github.com/Obelixor-Team/city-screensaver.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd city-screensaver
    ```
3.  Build and run the application in release mode:
    ```bash
    cargo run --release
    ```

## Controls

-   Press any key to exit the screensaver.
